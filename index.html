Mandelbrot Set

The Mandelbrot set is one of the most popular fractals in mathematics. It is named after Benoit Mandelbrot, a Polish-French-American mathematician who, in year 1980, obtained high quality visualisations of the set. The Mandelbrot set can be explained with the equation Zn+1 = Zn2+c, where "Z" and "c" are complex numbers and "n" is a positive integer. Starting with Z0 = 0, "c" is in the Mandelbrot set if the absolute value of Zn never becomes larger than a certain threshold, no matter how large "n" gets. The Mandelbrot set can be represented with colours, each point being coloured based on how fast it goes towards infinity. The final result is an image of the entire set, which can be explored by zooming into it.

Julia Set

The Julia set is a fractal that is closely related to the Mandelbrot set. It is named after the French mathematician Gaston Julia, who investigated its properties in 1915 and released a famous paper in 1918. Applying the same formula as in the Mandelbrot set, Zn+1 = Zn2+c, results in interesting fractal images that can be explored. The difference between the two sets is what numbers within the formula are changed. To obtain Mandelbrot, the value of Z0 starts at 0 and the value of "c" is taken from the complex plane. For the Julia set, the complex number "c" is constant and Z0 takes values from the complex plane. To change how a Julia set fractal looks, a different complex number can be assigned to "c".

Perlin Noise

Perlin noise is a powerful algorithm that is used often is procedural content generation. It is mostly use by visual effects artists to increase the appearence of realism in computer graphics. The algorithms was developed by Ken Perlin in 1983, as the result of his frustration with the "machine-like" look of computer-generated imagery at the time. Compared to generating totally random patterns, Perlin noise creates values that change more smoothly. When applied to textures and computer art, the resultant image has a much more "natural" and "organic" look.
