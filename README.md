# Procedurally-generated-art-gallery
Individual project for 3rd year
Author: Septimiu-Iulian Calin
K-number: k19010784

Pre-requisites for running the application:
- The latest version of Visual Studio Code is installed
- The Live Server extension is installed in Visual Studio Code

Instructions for running the application:
- Open the project folder with Visual Studio Code
- On the Explorer tab, right-click the index.html file and select the Open with Live Server option
- The application should open in a new tab on the web browser
